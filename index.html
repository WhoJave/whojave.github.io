MoJave安装iTunes12.6下载ipa

发表于 2019-02-26

iTunes 12.6 带有 App Store 功能，可以下载和管理 App 的 ipa 安装包。

  1. 下载这个文件，双击打开文件，或者使用“脚本编辑器”打开这个文件 (文件下载地址：http://t.cn/E4tLunQ )
  2. 脚本编辑器，点击“运行”按钮(或按快捷键 Command+R)，根据提示操作就行了
  3. 大致过程：先删除本机 iTunes，选择 iTunes 12.6 文件，修改文件后再安装 iTunes 12.6
  4. 方法来源：http://t.cn/E4txIvq
  5. 如果打开 iTunes 12.6 提示无法读取文件“iTunes Library.itl”，因为它是由更高版本的 iTunes 所创建。，点击“退出”按钮，打开文件夹 /用户/cc/音乐/iTunes(cc是用户名)，删除 “iTunes Library.itl” 文件，再 iTunes 12.6 就 OK 了

iTunes 下载地址：http://t.cn/E4tAFuY

Ubuntu中vi命令的使用

发表于 2018-09-18

在Ubuntu系统中,进入vi命令编辑模式无法使用方向键移动光标以及Mac键盘中Delete无法正常删除字符.

这是由于Ubuntu系统中的vi是vim-tiny版本,使用方法可以在终端输入man vi 然后按下h 即可.(参看下图或跳过看安装vim步骤)

移除vim.tiny

sudo apt-get remove vim-common

安装vim.full版本

sudo apt-get install vim

ios-class-guard混淆代码基本使用

发表于 2018-08-08

GitHub 点我

Homebrew

ios-class-guard是在class-dump的基础进行的混淆处理

  • 安装 (将安装在/usr/local/bin目录下，安装后可以在Terminal使用which ios-class-guard 命令查看)

    或安装bleeding edge version(测试版)：

  • 使用

    1.下载 obfuscate_project 到项目根目录

    2.使用终端或其他文本编辑工具打开obfusacate_project 修改PROJECT，SCHEME，CONFIGURATION，SDK等

    3.按照MyProject-Prefix.pch 格式创建PCH文件（iOS-Class-Guard会自动查找改pch文件并引入后续生成的symbols.h）

    • 步骤

      1.XCode File -> New -> File -> iOS -> Other -> PCH File

      2.在target的Build Setting 中设置 Prefix Header 路径 并设置 Precomplie Prefix Header 为 YES

    4.终端执行bash obfuscate_project (注意：如果在git环境下先commit一次代码，因为执行此命令会放弃本地所有更改，如果不在git下直接Go2Shell到终端生产Git环境

    执行bash obfuscate_project 若成功则为在路径下生成 symbols.h 与 symbols.json 若失败则编译项目，查看报错的地方，在obfuscate_project 中添加忽略项 eg: CLASS_GUARD_OPTS= “-i width” 表示不混淆width 修改完后记得commit obfuscate_project 文件

验证：

编译成功后，Product下出现xx.app,在Finder中显示包内容，将可执行文件拷贝到某路径下，在目录创建一个文件夹存放头文件

class-dump -H 可执行文件名 -o 输入目录

砸出的头文件不规则乱码这种表示混淆成功

本文是ios-class-guard的基本使用，后续将进一步剖析，敬请期待!
